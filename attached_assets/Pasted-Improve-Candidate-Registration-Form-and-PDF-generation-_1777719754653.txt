1. Training Centre ID / TC ID
- Add TC ID input option in Company/Center Settings or Candidate Registration Form.

2. Candidate ID Auto Fill
- Candidate ID should automatically fill with Candidate Aadhaar Number.
- Candidate ID = Aadhaar Number.
- Show Aadhaar number digit-by-digit in Candidate ID boxes.
- Candidate ID should be read-only.
- If Aadhaar number changes, Candidate ID should update automatically.

3. Caste Certificate Availability Flow
When candidate selects caste/category:
- SC
- ST
- OBC
- Other

Show question:
“Caste Certificate Available?”
Options:
- Yes
- No

If Yes:
- Continue existing upload process.
- Staff uploads caste certificate.
- PDF Page 5 = Uploaded Caste Certificate.

If No:
- Show option:
  “स्वघोषणा पत्र / Self Declaration for Caste Certificate”
- Show OK button.
- After OK, show input field:
  “जाति का नाम / Caste Name”
- Staff enters caste name and clicks OK/Confirm.
- Then generate Self Declaration page automatically using candidate registration data.

4. Final PDF Structure
PDF must always be exactly 5 pages:

Page 1 → Registration Form
Page 2 → Aadhaar Card (Front top, Back bottom on same page)
Page 3 → Education Certificate
Page 4 → Bank Passbook
Page 5 → Conditional:
   - If Caste Certificate Available = Yes:
     Show uploaded Caste Certificate
   - If Caste Certificate Available = No:
     Generate Self Declaration for Caste Certificate page
