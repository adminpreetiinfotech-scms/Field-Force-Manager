# Nistha Skill Field Staff Manager — User Guide
**Version 1.0.2 | DDU-KK/JSDMS, Jharkhand**

---

## ADMIN PANEL (Web)

### Login
1. Open the Admin Panel in your browser
2. Enter your **10-digit phone number** and **4-digit MPIN**
3. Click **Login**

### Dashboard
- View real-time stats: total staff, active today, candidates registered
- Leaderboard: top performing staff by candidate count
- Live map: see where field staff are right now

### Staff Management
| Action | How |
|--------|-----|
| View all staff | Staff → Staff List |
| Approve/Reject new staff | Staff → Pending Approvals |
| Disable a staff member | Staff → Click staff → Disable |
| Edit profile | Staff → Click staff → Edit Profile |

### Candidates
| Action | How |
|--------|-----|
| View all candidates | Candidates tab |
| Filter by status | Use Status dropdown |
| Change status (shortlisted/selected/rejected) | Click candidate → Change Status |
| Download PDF | Click candidate → Download PDF |
| Export all as CSV | Candidates → Export CSV |

### Notices
- Send notices to all staff or specific groups
- Staff receive instant notification on their mobile app
- SMS is also sent automatically (up to 50 recipients)

### Reports
- View daily/monthly activity reports
- Filter by staff member, date range
- WhatsApp share: one-click share for weekly report

## FIELD STAFF APP (Mobile)

### First-Time Login
1. Open the app → Enter **10-digit phone number**
2. Enter your **4-digit MPIN** (set during registration)
3. You're in!

### Daily Workflow
```
Start Shift → Check In (selfie + GPS) → Record Activities → Check Out
```

#### Check In
1. Tap **Check In** on the shift screen
2. Allow camera and location access
3. Take a selfie (live capture only)
4. GPS auto-stamps your location
5. Shift timer starts

#### Recording Trips
1. Tap **Start Trip** to begin
2. GPS tracks your route automatically
3. Tap **End Trip** when done
4. Km is auto-calculated

#### Meter Reading
1. Tap **Add Meter Reading**
2. Take a photo of the meter
3. Enter reading value
4. GPS stamps location

#### Check Out
1. Tap **Check Out**
2. Take a selfie to confirm
3. Daily km summary shown

### Candidate Registration
1. Go to **Register Candidate** from home
2. Fill in all required fields
3. Take candidate's **Aadhaar photo** (front)
4. Take candidate's **passport photo**
5. Submit — PDF is auto-generated

**Tip:** System checks for duplicate Aadhaar/phone before submitting.

### Attendance Calendar
- Go to **Profile → Attendance** to see your monthly calendar
- 🟢 Green = Full Day Present (check-in + check-out)
- 🟡 Yellow = Partial Day (check-in only)
- ⚪ Grey = Absent

### Notifications
- Bell icon shows unread notices from admin
- Tap to read notice details

## REGISTRATION FLOW (New Staff)

1. Staff downloads app → Tap **Register**
2. Fill name, phone, area details
3. Enter **Admin Code** (get from your manager)
4. Set 4-digit MPIN
5. Wait for admin approval
6. Once approved, login with phone + MPIN

## TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| Can't check in | Allow Location and Camera in phone settings |
| GPS not tracking | Go outside, ensure location is enabled |
| App shows old data | Pull down to refresh |
| Candidate submission fails | Check internet connection, try again |
| Forgot MPIN | Contact your admin |
| Login failed | Check phone number (10 digits, no +91) |
